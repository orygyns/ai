Problem & Opportunity

What students do today

Students paste notes into ChatGPT, get generic responses optimized for no one, then sift through irrelevant output. There is no understanding of their syllabus, no PYQ context, no exam format awareness. Generic AI costs ₹1,600+/month. Most students never pay — and get nothing.

What ORYGYNS AI does

A focused study layer purpose-built for exam preparation. PDF intelligence that reads their actual notes. A PYQ repository that surfaces historically important questions. Revision tools calibrated to 2-mark, 5-mark, and 10-mark formats. Free to use with their own Groq key — which costs them near nothing.

Positioning
ORYGYNS AI sits at the intersection of three things no single product currently offers: general AI study chat (ChatGPT) + document intelligence (NotebookLM) + exam-specific preparation tools (none exist at scale). Our moat is the PYQ repository, the domain-tuned prompt system, and the compounding semantic cache — which gets cheaper and smarter with every student interaction.

V1 Feature Modules
Core

AI Study Chat

Ask any academic question, get concept explanations, generate study notes, request first-year or expert-level answers. Powered by Groq's 8B/70B models with exam-aware prompts.

Core

PDF Intelligence

Upload notes, textbooks, PPTs, lab manuals. Chat with documents, extract key points, generate chapter summaries. RAG pipeline — only relevant chunks sent to model, never full files.

Unique

PYQ Repository

Admin-verified previous year questions. Topic-wise and subject-wise search. Pattern recognition across years. PYQ-based mock tests. Disclaimer: frequency-based importance, not guaranteed prediction.

Generate 2-mark, 5-mark, 10-mark questions. Viva questions. Quick revision sheets. One-page summaries. Output format matches actual exam structure — not generic Q&A.

Revision

Convert uploaded notes or chat context into Question → Answer flashcard decks. Built for last-minute revision and memory retention. Exportable for offline use.

Assessment

Generate MCQs, short-answer, and subject/topic quizzes at Easy, Medium, or Hard difficulty. Students can test themselves directly in the platform. Instant grading.

One-click ready-made prompts for common tasks: Summarize Notes, Exam Prep, Quick Revision, Flashcards, Concept Explanation. Reduces friction for students new to AI tools.

AI Quality Modes
Low
llama-3.1-8b-instant
Fastest. Least tokens. Best for definitions, quick summaries, short explanations.
Medium
Auto-routed
Default mode. Balanced quality and speed. Handles most student tasks efficiently.
High
llama-3.3-70b-versatile
Better reasoning. Long-form answers. Complex concepts, research, nuanced questions.
Max
Best available
Highest reasoning. Deep analysis, advanced academic queries, difficult problem sets.

Medium is the default. The router automatically uses 8B (80% of queries) unless complexity signals require 70B. Students can override at any time.

Request Pipeline
01 Semantic Cache Query embedded via all-MiniLM-L6-v2 (22MB, CPU). Cosine similarity ≥ 0.92 → return cached answer. No Groq call. $0.00 · you host
02 LLMLingua-2 Compress prompt 4–10× via Microsoft's open-source CPU model before it reaches Groq. Preserves meaning. $0.00 · you host
03 History Compactor After 5 turns, summarize history to a 50-token digest. Replaces 300–500 tokens of raw context. ~$0.000002
04 Simple Router Token count + keyword check. <60 tokens or simple task → 8B. ≥60 or code/math keywords → 70B. $0.00 · you host
05 Groq: 8B (80%) llama-3.1-8b-instant. Fast definitions, summaries, short-form prep. 500–1000 TPS. max_tokens=200. $0.05/M · user key
06 Groq: 70B (20%) llama-3.3-70b-versatile. Complex explanations, research, high-quality study content. $0.59/M · user key
07 Cache Write Response vector stored async in pgvector. Future identical queries answered instantly for free. Async · $0.00

─ Sky = you host  |  ─ Grey = Groq Cloud (student's API key pays). No fallback providers.

Key storageStudent pastes Groq key once. Stored AES-256-GCM encrypted in Supabase. Never logged in plaintext. Displayed masked.
